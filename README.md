# WeatherForecast
iOS Weather Forecast application allowing to show weather in the city we've entered.

#### Features
- Take a city
- Show the weather

#### API Used:
- [RecastAI](https://cai.tools.sap/)
- [DarkSkyKit](https://darksky.net/dev/docs)
